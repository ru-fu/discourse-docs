Install Anbox Cloud
###################

There is a difference between Anbox Cloud and Anbox Cloud Appliance, see MISSING EXPLANATION. You should start with the appliance, and then move on to a full installation for your production setup.

To install the appliance, follow the :doc:`tutorial <install-appliance>`.

The following topics are about the full installation of Anbox Cloud.

Before you install, check the :doc:`requirements`.
